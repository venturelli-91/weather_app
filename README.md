## 📝 Descrição

Um **Aplicativo de Clima** moderno e intuitivo construído com **Next.js**, **React**, **TypeScript**, **TailwindCSS** e **Flowbite**. Esta aplicação permite aos usuários verificar a previsão do tempo atual de qualquer cidade do mundo, buscando dados de uma API de clima.

O aplicativo implementa **Variáveis de Ambiente** para gerenciamento seguro de chaves de API, demonstrando as melhores práticas para lidar com informações sensíveis em aplicações em produção. Seja para planejar seu dia, verificar condições para viagem, ou simplesmente ter curiosidade sobre o clima ao redor do mundo, esta aplicação fornece informações meteorológicas precisas e atualizadas com uma interface limpa e amigável.

## 📌 Funcionalidades

- **Clima Atual**: Exibe temperatura, umidade e condições climáticas para qualquer cidade.
- **Pesquisa de Cidades**: Os usuários podem pesquisar e visualizar dados meteorológicos por nome da cidade.
- **Design Responsivo**: Layout otimizado para dispositivos móveis e desktop.
- **Ícones de Clima**: Indicadores visuais de clima (ex: ensolarado, nublado, chuvoso).
- **Framework Next.js**: Aproveita o poder do Next.js para renderização do lado do servidor e desempenho otimizado.
- **Componentes Flowbite**: Utiliza os componentes UI do Flowbite para uma interface de usuário refinada.
- **Variáveis de Ambiente**: Implementa manipulação segura de chaves de API e configuração usando arquivos .env.

## 🏃‍♂️ Como Executar o Projeto

1. Clone este repositório:

   ```bash
   git clone https://github.com/venturelli-91/weather_app.git
   ```

2. Navegue até o diretório do projeto:

   ```bash
   cd weather_app
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

5. Acesse o aplicativo em http://localhost:3000.

## 🌟 Funcionalidades Adicionais

- **Dados Climáticos em Tempo Real**: Busca dados ao vivo de uma API de clima para exibir informações atualizadas.

- **Tratamento de Erros**: Exibe uma mensagem de erro para nomes de cidades inválidos ou falhas nas requisições à API.

- **Modo Escuro**: Opção para alternar entre temas claro e escuro.

- **Interface Amigável ao Usuário**: Design simples e intuitivo para uma experiência de usuário sem complicações.

- **Gerenciamento Seguro de Chaves de API**: Utiliza variáveis de ambiente para armazenar e acessar chaves de API com segurança, seguindo as melhores práticas do setor.

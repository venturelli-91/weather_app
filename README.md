## 📝 Description

A modern and intuitive **Weather App** built with **Next.js**, **React**, **TypeScript**, **TailwindCSS**, and **Flowbite**. This application allows users to check the current weather forecast of any city worldwide by fetching data from a weather API.

The app implements **Environment Variables** for secure API key management, demonstrating best practices for handling sensitive information in production applications. Whether you're planning your day, checking conditions for travel, or simply curious about the weather around the world, this application provides accurate and up-to-date weather information with a clean, user-friendly interface.

## 📌 Features

- **Current Weather**: Displays temperature, humidity, and weather conditions for any city.
- **Search for Cities**: Users can search and view weather data by city name.
- **Responsive Design**: Optimized layout for mobile and desktop devices.
- **Weather Icons**: Visual weather indicators (e.g., sunny, cloudy, rainy).
- **Next.js Framework**: Leverages the power of Next.js for server-side rendering and optimized performance.
- **Flowbite Components**: Utilizes Flowbite's UI components for a polished user interface.
- **Environment Variables**: Implements secure handling of API keys and configuration using .env files.

## 🏃‍♂️ How to Run the Project

1. Clone this repository:

   ```bash
   git clone https://github.com/venturelli-91/weather_app.git
   ```

2. Navigate to the project directory:

   ```bash
   cd weather_app
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Access the app at http://localhost:3000.

## 🌟 Additional Features

- **Real-time Weather Data**: Fetches live data from a weather API to display up-to-date information.

- **Error Handling**: Displays an error message for invalid city names or failed API requests.

- **Dark Mode**: Option to toggle between light and dark themes.

- **User-friendly Interface**: Simple, intuitive design for a seamless user experience.

- **Secure API Key Management**: Uses environment variables to securely store and access API keys, following industry best practices.
